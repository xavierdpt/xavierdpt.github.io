URI: Uniform Resource Identifier

Uniform Resource Identifiers are defined in RFC3986.

More information is also available on Wikipedia

In this page, we will generate a variety of URI strings by following the rules of the grammar.

The root production rule is the following:

scheme:hier-part[?query][#fragment]

Let's expand the scheme rule.

ALPHA*(ALPHA/DIGIT/+/-/.):hier-part[?query][#fragment]

This means that the scheme must start with a letter, but then can continue with letters, digits, or '+', '-' or '.' characters.

For example, http2+ftp-1-5.x. would be a valid scheme according to this specification.

Let's use myscheme for now, which is equally valid.

myscheme:hier-part[?query][#fragment]

Since query and fragment are optional, we can ignore them for now.

myscheme:hier-part
