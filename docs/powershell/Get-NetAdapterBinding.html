Get-NetAdapterBinding

Get-NetAdapterBinding

ComponentClassName is a string.

Example values:

  • Client
  • Filter
  • Service
  • Transport

ComponentID and ElementName are strings.

Example values grouped by ComponentClassName:

  • Client
    • ms_msclient
  • Filter
    • INSECURE_NPCAP
    • ms_pacer
    • oracle_VBoxNetLwf
  • Service
    • ms_server
  • Transport
    • ms_implat
    • ms_lldp
    • ms_lltdio
    • vms_pp
    • ms_rspndr
    • ms_tcpip
    • ms_tcpip6

Name is the name of the adapter.

Except particular cases, all components are bound to all interfaces.

Example value: VirtualBox Host-Only Network

InterfaceDescription is the description of the adapter.

Example value: VirtualBox Host-Only Ethernet Adapter

Description and DisplayName are the localized description string of the component.

Example values (in French) for each component

Component Description
ms_msclient Client pour les réseaux Microsoft
INSECURE_NPCAP Npcap Packet Driver (NPCAP)
ms_pacer Planificateur de paquets QoS
oracle_VBoxNetLwf VirtualBox NDIS6 Bridged Networking Driver

ms_tcpip Protocole Internet version 4 (TCP/IPv4)
ms_tcpip6 Protocole Internet version 6 (TCP/IPv6)
vms_pp Commutateur virtuel extensible Hyper-V

ComponentClassGuid is the GUID of the component

Example value: {4D36E973-E325-11CE-BFC1-08002BE10318}

InstanceID is the component name qualified with the GUID of the adapter.

Example value: {545C58EC-A1C2-468B-ADD7-00E3A9604DA2}::ms_tcpip

Caption is the name bniding class followed by the name of the adapter.

Example value: MSFT_NetAdapterBindingSettingData 'VirtualBox Host-Only Ethernet Adapter'

Source is always 1, possibly unless you do funny things.

Enabled is always false for ms_implat and vms_pp, always true for the others components.

BindName does not always have a value. Here are the values for the components that have one.

Component Bind Name

ms_server LanmanServer

vms_pp VMSP

Characteristics is dependent on the adapter, and the same for all components

Example values: 1, 132, 161

SystemName is the host name

PSComputerName is always blank
