Get-NetAdapterChecksumOffload

Get-NetAdapterChecksumOffload

This function gets some of the advanced properties available in Get-NetAdapterAdvancedProperty.

Main object

Name is the name of the adapter

Example value: vEthernet (Default Switch)

Description, ElementName and InterfaceDescription are the description of the adapter

Example value: Hyper-V Virtual Ethernet Adapter

Caption is the description of the adapter prefixed with the class name.

Example value: MSFT_NetAdapterChecksumOffloadSettingData 'Hyper-V Virtual Ethernet Adapter'

InstanceID is the GUID of the adapter.

Example value: {DABC4D36-0A7F-4009-8CD9-D0A9337960B6}

Source is 2.

SystemName is the host name.

PSComputerName is blank.

ChecksumOffloadHardwareCapabilities is a CIM object.

Example value when rendered as a string: MSFT_NetAdapterChecksumOffloadCapabilities

IpIPv4Enabled, TcpIPv4Enabled, TcpIPv6Enabled, UdpIPv4Enabled, UdpIPv6Enabled are enum values of type NetAdapterChecksumOffload.Direction.

They have all this value: RxTxEnabled

NetAdapterChecksumOffload.Direction

Possible values for Microsoft.PowerShell.Cmdletization.GeneratedTypes.NetAdapterChecksumOffload.Direction:

  • Disabled
  • RxEnabled
  • RxTxEnabled
  • TxEnabled

Encapsulation

The following objects have the same properties with the same values:

  • IPv4ReceiveEncapsulation
  • IPv4TransmitEncapsulation
  • IPv6ReceiveEncapsulation
  • IPv6TransmitEncapsulation
$IPv4Receive = ($cap).IPv4ReceiveEncapsulation

They have more boolean properties, among with only NdisEncapsulationIeee802_3 is true, the others are false.

  • NdisEncapsulationIeeLlcSnapRouted
