Get-NetAdapterPowerManagement

Get-NetAdapterPowerManagement

Common properties

Name is the name of the adapter

ElementName, Description and InterfaceDescription are the description of the adapter

Caption is the interface description prefixed with MSFT_NetAdapterPowerManagementSettingData

InstanceID is the GUID of the adapter

Source is 2 or 3

SystemName is the hostname

PSComputerName is blank

OffloadParameters

$offloadParameters = (Get-NetAdapterPowerManagement).OffloadParameters

This contains an an array of objects of diverse types.

OffloadParameters : MSFT_NetAdapterPowerManagement_Offload_NS

FriendlyName is ND Offload

ID is an integer

MacAddress is a string

Example value: D4-BE-D9-8D-46-9A

OffloadType is an integer

Priority is a large integer

PSComputerName is blank

RemoteIPv6Address is string

Example value: ::

SolicitedNodeIPv6Address is string representing an IPv6 address

Example value: 2001:db8::1234:5678

TargetIPv6Addresses is a list of strings representing IPv6 addresses.

Example value: {2001:db8::1234:5678, ::}

OffloadParameters : MSFT_NetAdapterPowerManagement_Offload_Arp

FriendlyName is ARP Offload

HostIPv4Address is a string representing an IPv4 address

Example value: 192.168.0.1

ID is an integer

MACAddress is a string representing a MAC address

Example value: D4-BE-D9-8D-46-9A

OffloadType is an integer

Priority is a large integer

PSComputerName is blank

RemoteIPv4Address is a string representing an IPv4 address

Example value: 0.0.0.0

WakePatterns

$wakePatterns = (Get-NetAdapterPowerManagement).WakePatterns

This contains an an array of objects of diverse types.

WakePatterns : MSFT_NetAdapterPowerManagement_WakePattern_MagicPacket

FriendlyName is a string, typically Magic Packet

ID is an integer

Priority is an integer

PSComputerName is blank

WakePacketType is an integer

WakePatterns : MSFT_NetAdapterPowerManagement_WakePattern_TcpSyn

FriendlyName is a string, usually blank

ID is an integer

Priority is an integer

PSComputerName is blank

WakePacketType is an integer

DestinationAddress is an string that can represent an IPv4 or an IPv6 address.

Example values: ::, 0.0.0.0

DestinationPort is an integer that can be 0

SourceAddress is an string that can represent an IPv4 or an IPv6 address.

Example values: ::, 0.0.0.0

SourcePort is an integer that can be 0

WakePatterns : MSFT_NetAdapterPowerManagement_WakePattern_Bitmap

FriendlyName is a string

Example values:

  • TCP PortCoalescedPattern
  • UDP PortCoalescedPattern
  • TCP V4 Packet
  • TCP V6 Packet
  • NCB:Inbox Client - WNS/EAS

ID is an integer

Priority is a possibly very large integer

PSComputerName is blank

WakePacketType is an integer, usually 1

Mask and Pattern are arrays of bytes

Settings

The following properties are of type NetAdapterPowerManagement.Setting:

  • AllowComputerToTurnOffDevice
  • ArpOffload
  • D0PacketCoalescing
  • DeviceSleepOnDisconnect
  • NSOffload
  • RsnRekeyOffload
  • SelectiveSuspend
  • WakeOnMagicPacket
  • WakeOnPattern

Possible values for Microsoft.PowerShell.Cmdletization.GeneratedTypes.NetAdapterPowerManagement.Setting:

  • Disabled
  • Enabled
  • Inactive
  • Unsupported
