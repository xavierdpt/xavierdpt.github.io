Get-NetAdapterLso

Get-NetAdapterLso

Properties common to other commands

Name is the name of the adapter.

ElementName, Description, InterfaceDescription is the description of the adapter.

Caption is the description of the adapter prefixed with MSFT_NetAdapterLsoSettingData.

InstanceID is the GUID of the adapter.

SystemName is the hostname.

PSComputerName is blank.

Source is 2.

Boolean and number properties

There are three boolean properties in this command:

  • IPv4Enabled
  • IPv6Enabled
  • V1IPv4Enabled

MaximumLsoVersionSupported is the only number property.

LargeSendOffloadV1HardwareCapabilities

$v1 = (Get-NetAdapterLso).LargeSendOffloadV1HardwareCapabilities

Two boolean properties

  • IPv4IpOptionsSupported
  • IPv4TcpOptionsSupported

Two number properties

  • IPv4IpOptionsSupported
  • IPv4TcpOptionsSupported

PSComputerName is blank

LargeSendOffloadV1HardwareCapabilities.IPv4Encapsulation

$v1Encapsulation = ($v1).IPv4Encapsulation

Boolean properties:

  • NdisEncapsulationIeee802_3
  • NdisEncapsulationIeee802_3pAndq
  • NdisEncapsulationIeee802_3PAndQInOob
  • NdisEncapsulationIeeLlcSnapRouted
  • NdisEncapsulationNotNull
  • NdisEncapsulationNotSupported

PSComputerName is blank

LargeSendOffloadV2HardwareCapabilities

$v2 = (Get-NetAdapterLso).LargeSendOffloadV2HardwareCapabilities

Two boolean properties

  • IPv6IpExtensionHeadersSupported
  • IPv6TcpOptionsSupported

Four number properties

  • IPv4MaxOffloadSizeSupported
  • IPv4MinSegmentCountSupported
  • IPv6MaxOffLoadSizeSupported
  • IPv6MinSegmentCountSupported

PSComputerName is blank
