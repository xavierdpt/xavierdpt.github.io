Get-NetAdapterAdvancedProperty

Get-NetAdapterAdvancedProperty

Network adapters can have custom properties defined for them

Properties

The Name property is the name of the adapter in which the property is defined

Example value: Wi-Fi

The RegistryKeyword property has the internal name of the advanced property, prefixed with "*" when it is a required property.

Example value: *DeviceSleepOnDisconnect

The InstanceID property is the GUID of the adapter with the internal name of the advanced property.

Example value: {D8332A15-3AB0-46F3-95FA-8EC5422775A7}::*DeviceSleepOnDisconnect

The Optional property tell whether the advanced property is mandatory or not.

The Description, DisplayName and ElementName have the same localized values.

Example value (in French): Veille à la déconnexion WoWLAN

The InterfaceDescription is the adapter's interface description.

Example value: Hyper-V Virtual Ethernet Adapter

The Caption is the interface description prefixed with the object's class name.

Example value: MSFT_NetAdapterAdvancedPropertySettingData 'Hyper-V Virtual Ethernet Adapter'

The RegistryDataType is always 1.

The DisplayDataType is 1, 5 or 6. It seems that 1 is for numbers

The NumericParameterBaseValue is always 10.

The NumericParameterStepValue is always 1.

The NumericParameterMinValue is 0, 1 or 100.

The NumericParameterMaxValue is 63, 64 or 1500.

The Source is always 3.

The RegistryValue and DefaultRegistryValue are numbers.

Example value for RegistryValue: {3}

Example value for DefaultRegistryValue: 3

The DisplayValue and DefaultDisplayValue property can be numbers, text strings or localized text strings.

Example value for DisplayValue and DefaultDisplayValue : 8, Enabled, 1. Aucune préférence, RTS/CTS activée

The ValidRegistryValues property contains list of numbers, exposed in a string type.

Example 1: {0,1,2,3}

Example 1: {1514,4088,9014}

The ValidDisplayValues property contains list texts, exposed in a string type.

Example 1: {Disabled, Tx Enabled, Rx Enabled, Rx & Tx Enabled}

Example 2: {Disabled, 4088 Bytes, 9014 Bytes}

The SystemName property has the host name as a string.

The PSComputerName property was always blank.
