Get-NetAdapterRdma

Get-NetAdapterRdma

Result type: Microsoft.Management.Infrastructure.CimInstance#ROOT/StandardCimv2/MSFT_NetAdapterRdmaSettingData

Common properties

Name is the name of the adapter.

ElementName, Description, InterfaceDescription is the description of the adapter.

InstanceId is the GUID of the adapter.

Caption is the description of the adapter prefixed with MSFT_NetAdapterRdmaSettingData.

PSComputerName is blank.

SystemName is the hostname.

Source is 2.

Number and boolean properties

Enabled is a boolean

The following properties are integers:

  • MaxCompletionQueueCount
  • MaxInboundReadLimit
  • MaxMemoryRegionCount
  • MaxMemoryWindowCount
  • MaxOutboundReadLimit
  • MaxProtectionDomainCount
  • MaxQueuePairCount
  • MaxSharedReceiveQueueCount

QosConfiguration

ETS and PFC are of type NetAdapterRdma.QosConfiguration

Possible values for Microsoft.PowerShell.Cmdletization.GeneratedTypes.NetAdapterRdma.QosConfiguration

  • False
  • NA
  • True

RdmaMissingCounterInfo

RdmaMissingCounterInfo is of type MSFT_NetAdapter_RdmaMissingCounterInfo

This type exposes the following boolean properties:

  • AcceptPerformanceCounterMissing
  • ActiveConnectionPerformanceCounterMissing
  • CompletionQueueErrorPerformanceCounterMissing
  • ConnectFailurePerformanceCounterMissing
  • ConnectionErrorPerformanceCounterMissing
  • ConnectPerformanceCounterMissing
  • RDMAInFramesPerformanceCounterMissing
  • RDMAInOctetsPerformanceCounterMissing
  • RDMAOutFramesPerformanceCounterMissing
  • RDMAOutOctetsPerformanceCounterMissing

RdmaAdapterInfo

$RdmaAdapterInfo = (Get-NetAdapterRdma).RdmaAdapterInfo
